LaTeX学习笔记

发表于2020-12-05|更新于2020-12-21|LaTeX

我从2020年暑期开始接触LaTeX\LaTeXLATE​X，真心觉得它不愧为写作利器，特别对于理工科的学生以及教师工作人员来说，LaTeX\LaTeXLATE​X极大地加快了写作效率。对于刚入门的同志们我在这里强烈建议阅读文档:lshort，想要阅读更多关于LaTeX\LaTeXLATE​X的教程，推荐在zlibrary上搜索下载，另外LaTeXstudio是一个不错的LaTeX\LaTeXLATE​X学习网站。

LaTeX须知

概述

TeX

TeX\TeXTE​X是高德纳(Donald E.Knuth) 开发的、以排版文字和数学公式为目的的一个计算机软件。高德纳从1977年开始开发TeX\TeXTE​X，以发掘当时开始用于出版工业的数字印刷设备的潜力。正在编写著作《计算机程序设计艺术》的高德纳，意图扭转排版质量每况愈下的状况，以免影响他的出书。我们现在使用的TeX\TeXTE​X排版引擎发布于1982年，在1989年又稍加改进以更好地支持8-bit字符和多语言排版。TeX\TeXTE​X以其卓越的稳定性、跨平台、几乎没有Bug 而著称。TeX\TeXTE​X的版本号不断趋近于π\piπ，当前为3.141592653。

LaTeX

LaTeX\LaTeXLATE​X为TeX\TeXTE​X基础上的一套格式，令作者能够使用预定义的专业格式以较高质量排版和印刷他们的作品。LaTeX\LaTeXLATE​X的最初开发者为Leslie Lamport博士。LaTeX\LaTeXLATE​X使用TeX\TeXTE​X程序作为自己的排版引擎。当下LaTeX\LaTeXLATE​X的主要维护者为Frank Mittelbach。
当前LaTeX\LaTeXLATE​X版本为LaTeX2ϵ\LaTeX2_{\epsilon}LATE​X2ϵ​，意思是超出了第二版，接近但是还没有达到第三版。

LaTeX的优缺点

优点

  • 专业的排版输出。
  • 方便而强大的数学公式排版能力，无出其右。
  • 绝大多数时候，用户只需要关注于一些组织文档结构的基础命令，无需操心文档的版面设计。
  • 很容易产生复杂的专业排版元素，如脚注，交叉引用，参考文献，目录等。
  • 强大的拓展性。世界各地的人开发了数以千计的LaTeX\LaTeXLATE​X宏包用于补充和拓展LaTeX\LaTeXLATE​X功能。
  • LaTeX\LaTeXLATE​X促使用户写出结构良好的文档。
  • LaTeX\LaTeXLATE​X依赖的TeX\TeXTE​X排版引擎和其它软件是跨平台，免费，开源的。

缺点

  • 入门门槛高。
  • 排查错误困难。
  • 样式定制困难。

LaTeX命令和代码结构

LaTeX命令和环境

LaTeX\LaTeXLATE​X命令以\开头，为以下两种形式之一：

  • 反斜线和后面的一串字母，如\LaTeX。它们以任意非字母符号(空格，数字，标点等)作为分隔符。
  • 反斜线和后面的一个非字母符号，如\$。它们无需分隔符。

要注意LaTeX\LaTeXLATE​X命令是对大小写敏感的。
大多数的LaTeX\LaTeXLATE​X命令带一个或多个参数，每个参数用花括号{和}包裹。还有些命令带一个或多个可选参数，以方括号[和]包裹。还有些命令在命令名称后可以带一个星号*，带星号和不带星号的命令效果有一些差异。
LaTeX\LaTeXLATE​X还引入了环境的用法，用以令一些效果在局部生效，或者生成特定的文档元素。LaTeX\LaTeXLATE​X环境的用法为一对命令\begin和\end:

引擎，格式和命令

在开始编译之前，有必要澄清几个概念：

  • 引擎 全称排版引擎，是读入源代码并编译生成文档的程序，如pdfTeX\TeXTE​X,\xeTeX等。有时也直接称为编译器。
  • 格式 是定义了一组命令的代码集。LaTeX\LaTeXLATE​X就是最广泛应用的一个格式，高纳德本人还编写了一个简单的plain TeX\TeXTE​X格式，没有定义诸如\documentclass和\section等等命令。
  • 命令 是引擎和格式二者的结合体。
